# Australian Standard Templates Specification

This document defines the requirements for Australian-compliant document templates.

---

## Invoice Template (Australian Tax Invoice)

### Legal Requirements (ATO)

For a document to be considered a valid **Tax Invoice** in Australia, it must contain:

| Field | Required | Description |
|-------|----------|-------------|
| "Tax Invoice" Label | ✅ Yes | Must clearly state "TAX INVOICE" |
| ABN | ✅ Yes | 11-digit Australian Business Number |
| Invoice Date | ✅ Yes | Date the invoice is issued |
| Seller Identity | ✅ Yes | Business name and contact |
| Buyer Identity | ✅ Yes | Customer name/address |
| Item Description | ✅ Yes | Description of goods/services |
| Quantity | ✅ Yes | Number of items |
| Unit Price | ✅ Yes | Price per unit (ex. GST) |
| GST Amount | ✅ Yes | 10% Goods and Services Tax |
| Total Amount | ✅ Yes | Including GST |

### Template Structure

```
┌─────────────────────────────────────────────────────────┐
│  [LOGO]                               TAX INVOICE       │
│                                                          │
│  Your Business Name                    Invoice #: 00001 │
│  ABN: XX XXX XXX XXX                   Date: DD/MM/YYYY │
│  123 Business Street                   Due: DD/MM/YYYY  │
│  Sydney NSW 2000                                        │
├─────────────────────────────────────────────────────────┤
│  BILL TO:                     SHIP TO:                  │
│  ┃ Customer Name              Customer Name             │
│  ┃ 456 Customer Road          456 Customer Road         │
│  ┃ Melbourne VIC 3000         Melbourne VIC 3000        │
├─────────────────────────────────────────────────────────┤
│  DESCRIPTION          QTY        RATE         AMOUNT    │
│  ─────────────────────────────────────────────────────  │
│  Service Item 1       1          $100.00      $100.00   │
│  Service Item 2       2          $50.00       $100.00   │
├─────────────────────────────────────────────────────────┤
│                                    Subtotal:  $200.00   │
│                                    GST (10%): $20.00    │
│                                    ───────────────────  │
│                                    TOTAL:     $220.00   │
├─────────────────────────────────────────────────────────┤
│  PAYMENT DETAILS                                        │
│  Bank: Commonwealth Bank                                │
│  Account Name: Your Business                            │
│  BSB: XXX-XXX                                          │
│  Account: XXXXXXXX                                      │
│                                                          │
│  Please include invoice number in payment reference.    │
└─────────────────────────────────────────────────────────┘
```

### Required Blocks

1. **Header Columns**
   - Left: Logo, Business Name, ABN, Address
   - Right: "TAX INVOICE", Invoice #, Date, Due Date

2. **Bill To / Ship To Columns**
   - Client Info block with left border accent

3. **Line Items Table**
   - Header row: Description, Qty, Rate, Amount
   - Dynamically add rows
   - Auto-calculate Amount = Qty × Rate

4. **Totals Table**
   - Subtotal
   - GST (10%)
   - Total (bold)

5. **Payment Details Block**
   - Bank name
   - Account name
   - BSB
   - Account number

## Implementation Notes

### GST Calculation

```javascript
const GST_RATE = 0.10; // 10% for Australia

const calculateGST = (subtotal) => subtotal * GST_RATE;
const calculateTotal = (subtotal) => subtotal + calculateGST(subtotal);
```

### ABN Validation

```javascript
// Simple format validation (XX XXX XXX XXX)
const formatABN = (abn) => {
    const digits = abn.replace(/\D/g, '');
    if (digits.length !== 11) return null;
    return `${digits.slice(0,2)} ${digits.slice(2,5)} ${digits.slice(5,8)} ${digits.slice(8,11)}`;
};
```

### Date Formats

Australian standard date format: **DD/MM/YYYY**

```javascript
const formatDateAU = (date) => {
    return date.toLocaleDateString('en-AU', {
        day: '2-digit',
        month: '2-digit',
        year: 'numeric'
    });
};
```
