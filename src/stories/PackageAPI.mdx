## üé® `EditorConfig` Object

The `config` prop allows you to tailor the editor's visual style and terminology.

### Theme Settings
```tsx
theme: {
  primaryColor?: string;   // Main brand color
  accentColor?: string;    // Hover states/highlights
  fontFamily?: string;     // Global editor font (e.g. 'Inter', 'serif')
  borderRadius?: string;   // UI component roundness
}
```

### Localization Labels
Override any internal UI text with the `labels` record.
```tsx
labels: {
  'Blocks': 'Component Library',
  'Page Properties': 'Document Setup',
  'Heading': 'Title Text'
}
```

## üìÑ `DocumentSchema`

The core data structure that holds your template.

```typescript
interface DocumentSchema {
  page: {
    size: 'A4' | 'LETTER' | 'LEGAL';
    margins: { top: number; right: number; bottom: number; left: number };
    padding: number;
    backgroundColor?: string;
    watermark?: WatermarkSettings;
  };
  elements: Record<string, EditorElement>;
  rootElementIds: string[];
}
```
